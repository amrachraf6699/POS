# Acceptance Criteria

- Environment configuration clearly distinguishes SQLite development/test from MySQL production.
- PHP/Laravel constraints are enforced and documented.
- `nwidart/laravel-modules` is installed at a verified Laravel 10/PHP 8.1-compatible version and module locations, test locations, and naming conventions are documented.
- Arabic locale, RTL direction, and base layout conventions are defined.
- Quality commands, package compatibility records, and incremental commit checkpoints are reproducible.
